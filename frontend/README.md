# Ogame UI

## Installation

* You need:
    * nodeJS v4.2.2
    * bower v1.6.8
    * gulp v3.9.0

* Clone repository

```
git clone https://github.com/tomekbielaszewski/Ogame.git
```

* Go to project directory

```
cd Ogame/frontend
```

* Install npm dependencies

```
npm install
```

* Install bower dependencies

```
bower install
```

* Run application with gulp

```
gulp
```

## More

* For more gulp tasks call help

```
gulp help
```
